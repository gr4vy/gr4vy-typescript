# QueryParamPeriod

Filters the results to the set of data aggregated over the
duration of the specified period.

## Example Usage

```typescript
import { QueryParamPeriod } from "@gr4vy/sdk/models/operations";

let value: QueryParamPeriod = "7-days";
```

## Values

```typescript
"1-day" | "7-days" | "30-days"
```
