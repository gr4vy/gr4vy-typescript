# RedirectPaymentMethodCreate

Create a transaction for an APM/LPM that requires a redirect.

## Example Usage

```typescript

let value: RedirectPaymentMethodCreate = {
  method: "paypal",
  country: "DE",
  currency: "EUR",
  redirectUrl: "https://example.com/callback",
};
```
