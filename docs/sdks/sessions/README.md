# Sessions
(*sessions*)

## Overview

The sessions APIs are used to facilitate user authentication for the Gr4vy
dashboard.

### Available Operations

* [logout](#logout) - Logout user

## logout

Ends a user's session by invalidating the used access token.

### Example Usage

```typescript
import { Gr4vy } from "@gr4vy/node";

async function run() {
  const sdk = new Gr4vy({
    bearerAuth: "<YOUR_BEARER_TOKEN_HERE>",
  });

  const result = await sdk.sessions.logout();

  // Handle the result
  console.log(result)
}

run();
```
