# AuditLogs
(*auditLogs*)

## Overview

Audit Logs provide an historic record of changes made to your Gr4vy instance.

### Available Operations

* [listAuditLogs](#listauditlogs) - List audit logs

## listAuditLogs

Returns a list of audit logs.

### Example Usage

```typescript
import { Gr4vy } from "@gr4vy/sdk";

const gr4vy = new Gr4vy({
  bearerAuth: "<YOUR_BEARER_TOKEN_HERE>",
});

async function run() {
  const result = await gr4vy.auditLogs.listAuditLogs({
    limit: 1,
    cursor: "ZXhhbXBsZTE",
    userId: "8724fd24-5489-4a5d-90fd-0604df7d3b83",
    action: "created",
    resourceType: "buyer",
  });

  // Handle the result
  console.log(result)
}

run();
```
